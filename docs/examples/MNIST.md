# MNIST Number Prediction Example
[Back to the table of contents](../README.md)

This example uses the MNIST dataset to predict a number that was hand written.

## CODE

The current code is at [mnist.happyml](../../happyml_repo/scripts/mnist.happyml). 

## Data
The data is from [Kaggle](https://www.kaggle.com/datasets/oddrationale/mnist-in-csv?resource=download). 
It is a list of handwritten numbers, and what number they are.

```happyml
create dataset mnist_train
       with header
       with expected label value          at 0
       with given    image pixels(28, 28) at 1
       using file://../happyml_repo/raw/mnist_train.csv

create dataset mnist_test
       with header
       with expected label value          at 0
       with given    image pixels(28, 28) at 1
       using file://../happyml_repo/raw/mnist_test.csv          
```


## Task
The task is to predict which number was handwritten from a 28x28 image.

```happyml
create task label predict_number 
with test mnist_test
with goal speed
using mnist_train     
```
